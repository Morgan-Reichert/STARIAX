## MindScope

**MindScope** permet aux utilisateurs de suivre leur état mental au quotidien, de passer des tests psychologiques standardisés et d'obtenir des analyses via un LLM tournant entièrement en local (WebLLM). Aucune donnée n'est envoyée à un serveur — confidentialité totale garantie.

**Technologies clés** : Next.js 14 · @mlc-ai/web-llm · Recharts · jsPDF · TypeScript · Tailwind CSS · PWA

---

## NightWatch

**NightWatch** est une app mobile-first (PWA) qui accompagne les soirées entre amis : suivi BAC en temps réel, scanner IA de bouteilles (Mistral Pixtral), classements, stories, localisation du groupe, boutique premium (Stripe). Une app iOS native Swift complète l'expérience.

**Technologies clés** : React 18 · Vite · Supabase · Stripe · Mistral API · Framer Motion · Swift 5.9 · iOS 17

## Challenger IA

Plateforme d'IA conçue pour challenger vos arguments et renforcer votre pensée critique. L'IA adopte différentes personas (Architecte, Fact-checker, Opposant) et niveaux d'intensité pour confronter vos idées avec rigueur.

**Technologies clés** : React 19 · Gemini API · Firebase · Tailwind CSS 4
